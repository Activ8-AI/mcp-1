## [1.0.0] - 2025-01-XX

### Added
- Initial release of Teamwork MCP Server
- HTTP and STDIO transport modes
- Bearer token and OAuth2 authentication support
- Comprehensive tool framework for Teamwork.com integration
- Production-ready logging, monitoring, and observability
- Read-only mode for safe operations
- Docker support with multi-architecture builds
- Comprehensive documentation and examples

### Features
- **Projects API**: Create, update, list, and manage projects
- **Tasks API**: Full task lifecycle management
- **Users API**: User and team management
- **Comments API**: Comment management and threading
- **Time Tracking**: Timer and time log functionality
- **Milestones**: Project milestone management
- **Tags**: Tagging and organization features
